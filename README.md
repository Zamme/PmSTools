# PmSTools (Postman Side Tools)
## Description
Tools for post services workers. (Spain firstly. Waiting for other countries post servicies info.)

![image](https://github.com/Zamme/PmSTools/blob/main/PmSTools/Resources/Images/postman_01.png)

## Platforms
Android (ready) and iOS (comming soon).

## Features
### Code 2 Barcode
Take a picture of an alphanumeric code and transform it to barcode.
### Location Autosearch
Take a picture of a written address and find it automatically in a map.
### Autoroute
Take pictures addresses and create an optimal route.

## Installation
### Android
Download current release apk and install it.
### iOS
Pending to deploy it.
